March 2020 - Present

Ambulance Detection
Cloud Computing Live Prototype

Due to the substantial population in India, it is challenging for the ambulance carrying a patient to cross the signal junction. So this idea will be implemented at the massive traffic junctions which avoid waiting for an ambulance till it turns comes.

The components required to build this system are ESp8266,NEO-6M GPS Module, and a cloud platform.

  • Esp8266 send the coordinates of Ambulance using NEO-6M Module.
  • The cloud platform receives Coordinates of Ambulance and will measure the distance from signal junction to Ambulance
  • If the Ambulance seems to be 150 meters from the junction, then another Esp8266 will manipulate the signals in a way that the path in which ambulance approaches could be clear.
  • IBM uses the db2 database. Node-Red can analyse the data received at IBM. Based on the result obtained at Node-Red, it generates results to Esp8266 at signals.

Technologies :
IBM cloud to get coordinates of Ambulance using Arduino IDE, Esp8266 to communicate with IBM Watson

WiFi Controlled Car using ESP8266
Arduino project

In this project, I have built a simple Robot (robotic car) that can be controlled over Wi-Fi Network i.e. the user inputs for direction of the movement of the Robot are provided through the WiFi (with the help of a simple HTML Page).

  • The ESP8266 Module is responsible for connecting to the WiFi Network acts as a server.
  • A simple HTML page is created and the browser which opens this web page acts as a client.
  • Whenever we click on the web page, the corresponding information will be transmitted to the Server (ESP8266).
  • This information is further received by Arduino and it controls the motors of the robot.
